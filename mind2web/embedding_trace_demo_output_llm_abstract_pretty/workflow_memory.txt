## exploretock / Restaurant workflow 1
Source: Mind2Web exemplar
Abstraction: llm
Website: exploretock
Domain: Travel
Subdomain: Restaurant
Goal pattern: Find available restaurants for a specific reservation type and location.
When to use: When the user wants to find restaurants on Tock with a specific reservation type (e.g., Pickup, Delivery) and location.
When not to use: When searching for a specific restaurant by name or for a standard 'Dine in' reservation without changing the default location.
